Remove noopener and noreferrer (rel="noopener noreferrer") automatically

== Description ==

This plugin removes rel="noopener noreferrer" from your links. The plugins will remove this tag from any new link you create or add in your content. You may read <a href="https://udinra.com/blog/remove-rel-noopener-noreferrer-site">this article</a> to understand impact of this tag on your website. It will help you decide if you want to remove this tag from your site.

Note there is no configuration option. Just install the plugin it works.

Why remove rel="noopener noreferrer"?

This tag creates problem with Affiliate Network by hiding referrer information. So if you are facing the issue then install this plugin. You can also read the article I linked in description of this plugin to understand impact of this tag on your site.
